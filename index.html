jQuery.suggest

jQuery.suggest , a inline jQuery plugin. (You know the effect from Google's main search field.)

Example

Markup


<input type="text" name="search" id="search" class="search" />

Javascript


var haystack = ["ActionScript", "AppleScript", "Asp", "BASIC", ...];

$(function(){
  $('#container').suggest(haystack);
})
